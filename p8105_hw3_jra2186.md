2.  Make a plot that shows the number of items ordered in each aisle,
    limiting this to aisles with more than 10000 items ordered. Arrange
    aisles sensibly, and organize your plot so others can read it.

``` r
instacart %>%
  pull("aisle") %>%
  janitor::tabyl() %>%
  arrange(desc(n)) %>%
  filter(n > 10000) %>%
  ggplot(aes(x = ., y = n)) +
  geom_boxplot()
```
